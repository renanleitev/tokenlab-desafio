# Teste técnico TokenLab - Frontend

## Tecnologias utilizadas

1. React
2. TypeScript
3. Axios
4. Bootstrap
5. Redux

## Instalação

Execute o comando:

    npm i

Para rodar o site:

    npm run dev

Ou:

    npm start

Para acessar o site:

    http://localhost:5173/

Obs: O site irá abrir por padrão na porta 5173
